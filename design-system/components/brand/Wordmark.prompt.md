The wordmark lock-up — the Mark paired with the "MoneyBin" name. Composes `Mark` and bakes in the two things that are easy to get wrong: the optical baseline nudge on the text, and gilt on "Bin". Use a size preset; switch `bin="mono"` when another gold element shares the same bar or header.

```jsx
<Wordmark size="nav" />              // gold "Bin", nudge baked in
<Wordmark size="hero" bin="mono" />  // marketing, monochrome
```
