The logo mark. Pair with the wordmark set in Newsreader semibold. Use `plate="light"` on dark surfaces, `plate="dark"` on light.

```jsx
<div style={{display:'flex',alignItems:'center',gap:14}}>
  <Mark size={44} plate="light" />
  <span style={{fontFamily:'var(--font-display)',fontWeight:600,fontSize:24,color:'var(--text-primary)'}}>MoneyBin</span>
</div>
```
